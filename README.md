# InAmigos Foundation Website

This repository contains the source code for the **InAmigos Foundation NGO website**, built with HTML and CSS.  
The site showcases our mission, ongoing projects, and ways to get involved.

## 🌍 About InAmigos
InAmigos Foundation is a non-profit organization dedicated to empowering communities through:
- 📚 Education initiatives for underprivileged children
- 🏥 Healthcare camps and awareness programs
- 🌱 Environmental sustainability projects

## 🐾 Project Jeev
**Project Jeev** is our animal welfare initiative.  
It focuses on feeding, sheltering, and rescuing stray animals while spreading awareness about compassion and responsibility toward all living beings.

## 🚀 Features
- Responsive navigation bar
- Hero section with overlay text
- Project highlights (Education, Environment, Healthcare, Jeev)
- Call-to-action for volunteering and donations
- Footer with contact and social links
